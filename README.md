#Creating Classes
This lecture is on OOP using classes and methods. 

## Assignment 
The specifications for the assignment:
- Create a OOP calculator 
- It should include :
1) As a user I should be able to initialise a calculator object 
2) As a user I should be able to initialise the object with 2 numbers / arguments
3) As a user I should also be able to initialise a calculator object with NO numbers / arguments 
4) As a user I should be able to do the following operations: Add, Subtract, Multiply, Divide, get the area of a circle

The project has other specifications.It needs:
1) A README.md file to explain the project
2) Needs to have version control (git)
3) Follow seperation of concerns
